TestDataSupplier by gezope

WordPress plugin which provides lots of data for testing

Dependencies

It's a WordPress plugin so it only works with an installed WordPress.

Install

Regular installation needed. Upload the files under /wp-content/plugin subdirectory, then activate it from Admin's Dashboard.
